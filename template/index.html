Welcome to R universe!

We are running a pilot based on selected GitHub users and organizations that have existing CRAN packages.

# Install 'magick' from 'ropensci' universe
install.packages('magick', repos = 'https://ropensci.r-universe.dev')

# Install 'curl' from 'jeroen' universe
install.packages('curl', repos = 'https://jeroen.r-universe.dev')

Or enable a universe globally:

# Enable the 'ropensci' universe
options(repos = c(ropensci = 'https://ropensci.r-universe.dev', getOption("repos")))

# Install some packages
install.packages(c("magick", "pdftools", "drake"))

The source code for all packages in all universes is tracked on https://github.com/r-universe.
